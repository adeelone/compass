# Code Of Conduct

This project follows the Contributor Covenant v2.1.

Participants are expected to use welcoming and inclusive language, respect differing viewpoints, accept constructive feedback, and focus on what is best for the community.

Unacceptable behavior includes harassment, discriminatory language, trolling, publishing private information without permission, or other conduct inappropriate in a professional setting.

Maintainers may remove comments, commits, code, issues, or other contributions that do not align with this Code of Conduct.
