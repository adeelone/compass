# Contributing

## Setup

```bash
cp .env.example .env
make install
make test
```

## Branches

Use `feat/`, `fix/`, or `chore/` prefixes.

## Commits

Use Conventional Commits such as `feat(resume): add ats rule`.

## Pull Requests

Run lint, typecheck, tests, and eval before requesting review. Include screenshots for UI changes and note any provider or privacy impact.
