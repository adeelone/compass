# Scoring

## ATS Readability

Current scaffold rules:

- Contact email parses cleanly.
- Standard resume sections are present.
- Table-like spacing is not detected.
- Filename is short and recruiter-friendly.

Each rule returns pass or fail, severity, fix guidance, and source.

## Content Quality

Content scoring rewards quantified bullets and strong action verbs. Findings identify weak bullets and propose concrete rewrites.

## Match Score

The match score combines skill overlap, location fit, level fit, and a base compatibility score. Every reason cites the evidence input.
