# Security

## Reporting

Please report vulnerabilities privately to the maintainers. Do not open a public issue for security-sensitive findings.

## Supported Versions

Only the latest `main` branch is supported during the pre-1.0 period.

## Response Expectation

The project aims to acknowledge reports within 5 business days and provide a remediation plan when the issue is confirmed.

## Secrets

Never commit `.env`, private keys, service-account files, Terraform state, or provider credentials.
